### <img src="https://api.iconify.design/tabler:server.svg?color=white#gh-dark-mode-only" width="18" height="18" alt="backend" /> Backend (FastAPI)
<img src="https://api.iconify.design/tabler:play.svg?color=white#gh-dark-mode-only" width="16" height="16" alt="run" /> Jalankan perintah berikut di terminal:

```bash
cd backend
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

```bash
npm install
npm run dev
```
